# 图书管理系统简介 java1051

## 技术栈

本项目采用以下技术构建：

- 后端：Spring Boot
- 前端：Vue
- 数据库：MySQL
- 数据访问：MyBatis
- UI框架：ElementUI

## 运行环境

- JDK 1.8
- Mysql 5.5
- IDEA
- Node.js
- Maven

## 系统角色

系统主要分为以下两种角色：

### 管理员

- **图书信息管理**：实现对图书的增删改查操作。
- **图书类型管理**：对图书类型进行增删改查。
- **用户管理**：包括搜索、新增、修改和查询用户。

### 用户

- **图书管理**：查询图书信息。

## 功能模块

系统包含但不限于以下功能模块：

- 图书信息管理
- 图书类型管理
- 用户管理
